Write snippets in Firefox

1. Install the Mozilla Firefox.

2. Open the Scratchpad from the Firefox menu: Tools\Web Developer\Scratchpad OR Shift+F4

Firefox has no console.log method by default. You have a few option here.

  • Use the window.console.log instead of console.log
  • Install the Firebug plugin and that will give you console.log
  • Write a console.log method

    var console = console || {};
    console.log = console.log || function(msg) {window.console.log("" + msg);};

View the output

To see the output from the console.log / window.console.log you will have to turn on Logging in the browser: Tools\Web Developer\Web Console
In the Logging dropdown menu select Log.

Write code in scratchpad and view the result in Firefox console.
