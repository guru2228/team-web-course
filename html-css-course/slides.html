HTML and CSS

HTML

The basic structure

<!DOCTYPE html>
<html lang="sv">
<head>
    <meta charset="utf-8">
    <title></title>
</head>
<body>

</body>
</html>

Some basic elements

  • headlines
  • paragraphs
  • divs
  • spans
  • images
  • links

Headlines

<h1>Heading level 1</h1>
    <h2>Heading level 2</h2>
        <h3>Heading level 3</h3>
            <h4>Heading level 4</h4>
                <h5>Heading level 5</h5>
                    <h6>Heading level 6</h6>

There are no more levels!

Headlines

  • Each document should have a logical heading structure that starts at level 1 and doesn’t skip any levels
  • The title of the document’s main content should be a level 1 heading
  • There should be only one level 1 heading in each document

Paragraphs

<p>This is a paragraph. Its use is fairly self-explanatory.</p>
                    
<p>This is another paragraph.</p>
  • You are able to control paragraph margins consistently with CSS
  • It semantically describes the text as what it is – a paragraph

Paragraphs

An example of how to NOT do it

This isn't really a paragraph
<br>
<br>
Just some free flying text separated by some line breaks

Divs

Divs are used to group content for layouting and positioning with CSS. 

<div>
    <h3>This is a heading</h3>
    <p>This is a paragraph.</p>
</div>

Divs could be nested. This is useful if you are using divs to layout a web page.

<div class="container">
    <div class="article">Main content area</div>
    <div class="aside">Secondary content area</div>
</div>

Spans

To make it possible to style and manipulate text or part of a text the <span> element is suitable.

<p>A text which is unstyled <span class="stand-out">except from this part</span></p>

Images

Used to display an image on a web page.

<img src="../image.png" alt="The Jayway logotype">
  • The src (source) attribute points out the URL to the image to display.
  • The required alt (alternative) attribute specifies an alternate text for an image, if the image cannot be displayed.

Links

A link, when clicked (or tapped), takes you to another document or to a section within the current document.

<a href="http://www.jayway.com">This is a link</a>

Links

The id attribute can be used to create a bookmark inside the HTML document.

<a id="detailed-info">This is a document internal anchor</a>

And here are two ways of linking to the anchor.

<a href="#detailed-info">Click to read more</a>
<a href="http://www.jayway.com/page-with-anchors.html#detailed-info">Click to read more on jayway.com</a>
