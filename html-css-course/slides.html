Some basic elements

  • headlines
  • paragraphs
  • divs
  • spans

Headlines

<h1>Heading level 1</h1>
    <h2>Heading level 2</h2>
        <h3>Heading level 3</h3>
            <h4>Heading level 4</h4>
                <h5>Heading level 5</h5>
                    <h6>Heading level 6</h6>

There are no more levels!

Headlines

  • Each document should have a logical heading structure that starts at level 1 and doesn’t skip any levels
  • The title of the document’s main content should be a level 1 heading
  • There should be only one level 1 heading in each document

Paragraphs

<p>This is a paragraph. Its use is fairly self-explanatory.</p>
                    
<p>This is another paragraph.</p>
  • You are able to control paragraph margins consistently with CSS
  • It semantically describes the text as what it is – a paragraph
